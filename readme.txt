专为Wordpress附件下载设计的插件


== Description ==

无聊把Wordpress的版权说明作成了 my download 插件。安装简单、轻巧迅速！

相关插件及任何问题可在我的插件发布页面反馈即： www.iiexe.com

如果您要支持我继续开发或者完善,请访问 www.iiexe.com 谢谢！


== Installation ==

直接在后台搜索 mydownload 在线安装启用即可，无需任何设置和代码修改！
